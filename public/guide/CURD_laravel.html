# CURD命令大全

EasyAdmin8-Laravel框架以内置快速生成CURD的命令, 包括控制器、视图、模型、JS文件。能够使开发者效率得到进一步提升。

# 常用命令

# 生成ea8_test_goods表的CURD
php artisan admin:curd test_goods
